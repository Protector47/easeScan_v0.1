# EaseScan_v0.1
**EaseScan_v0.1** can perform multiples task. You just have to enter the URL of the website.
EaseScan will perform these tasks:

1) Subdomain Scanning
2) Ports Scanning
3) Poodle Vulnerability Scanning
4) Heartbleed Vulnerability Scaninng

Usage: **`./easescan_v0.1sh [domain_name]`**
